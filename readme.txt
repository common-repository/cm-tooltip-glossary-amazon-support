Adds the Amazon products recommendation widgets to the CM Tooltip Glossary. 

== Description ==

This plugin is an extension of the [CM Tooltip Glossary](http://wordpress.org/plugins/enhanced-tooltipglossary/).  You can use this CM Tooltip Glossary Amazon Widgets to
display [Amazon associates widgets](https://widgets.amazon.com/) inside tooltips on your site while using your Amazon associate code. This will add to your WordPress site rich, interactive mini-applications that bring Amazon functionality and help you gain profits.
Once the plugin is activated Amazon product recommendations will be added to any term which was defined in your CM glossary and will be shown once your hover over the term.
